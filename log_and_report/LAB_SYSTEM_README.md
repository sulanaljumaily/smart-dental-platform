# نظام ربط المختبر مع ملف المريض - العيادة السنية الذكية

## نظرة عامة

تم تطوير نظام ربط متقدم بين المختبر وملف المريض يوفر إدارة شاملة لطلبات المختبر مع إمكانيات التتبع والإشعارات في الوقت الفعلي.

## الميزات الرئيسية

### 1. إدارة طلبات المختبر
- **طلب مختبر جديد**: واجهة سهلة لإنشاء طلبات مختبر مع جميع التفاصيل المطلوبة
- **تتبع الحالة**: مراقبة مستمرة لحالة كل طلب (في الانتظار، قيد المعالجة، جاهز للتسليم، تم التسليم)
- **شريط التقدم**: عرض مرئي لتقدم كل طلب مع نسب مئوية
- **سجل التحديثات**: تاريخ شامل لجميع التحديثات والمراحل

### 2. دليل المختبرات المعتمدة
- **قاعدة بيانات شاملة**: قائمة بجميع المختبرات المعتمدة مع التفاصيل الكاملة
- **نظام التقييمات**: تقييمات وآراء الأطباء لكل مختبر
- **مقارنة الأسعار**: مقارنة شفافة للأسعار بين المختبرات
- **معلومات الاتصال**: تفاصيل كاملة للوصول للمختبرات
- **التخصصات**: تصنيف المختبرات حسب التخصصات

### 3. نظام الإشعارات
- **إشعارات فورية**: تنبيهات فورية عند تحديث حالة الطلبات
- **أنواع الإشعارات**:
  - تحديث الحالة
  - طلب جديد
  - جاهز للتسليم
  - تأخير
  - حالة عاجلة
- **وضع القراءة**: تتبع الإشعارات المقروءة وغير المقروءة

### 4. لوحة تحكم المريض
- **إحصائيات شاملة**: عرض إجمالي الطلبات، الحالات، التكاليف
- **فلترة وبحث**: بحث متقدم وفلترة حسب الحالة، الفترة الزمنية
- **تصدير البيانات**: إمكانية تصدير البيانات والتقارير
- **عرض تفصيلي**: تفاصيل كاملة لكل طلب مع الملفات المرفقة

### 5. إدارة الملفات
- **رفع الملفات**: إمكانية رفع الأشعة، الصور، والوثائق
- **تصنيف الملفات**: تنظيم الملفات حسب النوع
- **الربط بالأسنان**: ربط الملفات بأسنان معينة
- **مشاركة آمنة**: مشاركة آمنة للملفات مع المختبرات

## مكونات النظام

### 1. ملف المريض المحدث (`PatientProfile.tsx`)
- **ربط المختبر**: واجهة متكاملة لطلب وإدارة طلبات المختبر
- **عرض الطلبات**: قائمة شاملة بجميع طلبات المختبر مع حالة كل طلب
- **تتبع فوري**: تحديثات في الوقت الفعلي لحالة الطلبات
- **إدارة الإشعارات**: مركز إشعارات متكامل

### 2. دليل المختبرات (`LaboratoryDirectory.tsx`)
- **قاعدة بيانات المختبرات**: معلومات شاملة عن جميع المختبرات المعتمدة
- **نظام البحث والفلترة**: بحث متقدم وفلترة حسب التخصص، الموقع، السعر
- **صفحات تفصيلية**: عرض مفصل لكل مختبر مع المراجعات والتقييمات
- **مقارنة المختبرات**: مقارنة شاملة بين المختبرات المختلفة

### 3. لوحة تحكم المريض (`PatientLabDashboard.tsx`)
- **لوحة تحكم شاملة**: عرض إحصائيات شاملة لطلبات المختبر
- **إدارة الطلبات**: عرض وإدارة جميع طلبات المختبر في مكان واحد
- **تتبع مفصل**: تفاصيل شاملة لكل طلب مع سجل التحديثات
- **تصدير البيانات**: إمكانية تصدير التقارير والبيانات

## الحالة والحالات

### حالات طلبات المختبر
1. **في الانتظار** (pending): تم إنشاء الطلب ولكن لم يبدأ العمل بعد
2. **قيد المعالجة** (in_progress): المختبر يعمل على الطلب
3. **جاهز للتسليم** (ready): تم الانتهاء والطلب جاهز للاستلام
4. **تم التسليم** (delivered): تم تسليم الطلب للمريض
5. **ملغي** (cancelled): تم إلغاء الطلب

### مستويات الأولوية
- **منخفض** (low): طلب عادي بأولوية منخفضة
- **متوسط** (medium): طلب عادي بأولوية متوسطة
- **عالي** (high): طلب عاجل يحتاج اهتمام
- **عاجل** (urgent): طلب عاجل جداً يحتاج اهتمام فوري

## استخدام النظام

### للمرضى
1. **عرض الطلبات**: انتقل إلى تبويب "طلبات المختبر" في ملف المريض
2. **إنشاء طلب جديد**: انقر على "طلب مختبر جديد" واملأ النموذج
3. **تتبع الطلبات**: راقب تقدم طلباتك من خلال شريط التقدم
4. **استقبال الإشعارات**: احصل على إشعارات فورية عند تحديث حالة الطلبات

### للأطباء
1. **اختيار المختبر**: راجع دليل المختبرات لاختيار الأنسب
2. **إنشاء طلب**: املأ تفاصيل الطلب بدقة
3. **متابعة التقدم**: راقب حالة الطلبات باستمرار
4. **التواصل مع المختبر**: استخدم بيانات الاتصال المتاحة
